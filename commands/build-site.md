argument-hint: ""

  - "Bash(tesserae project build-site:*)"

Build the static site at `.tesserae/site/` from the compiled graph. Includes the 3D graph view, per-node wiki pages, search index, and the bundled ask widget. Pair with `/tesserae:serve` to view locally.

!`tesserae project build-site`
